Codici sconto Italia

Raccolta di script Python open-source per il recupero automatico di codici sconto dai negozi online italiani. I dati vengono prelevati in tempo reale da Shopilo.it, il piu grande aggregatore di coupon in Italia.

Python 3.8+ MIT License npm disponibile Dati live shopilo.it
# Installa le dipendenze
pip install requests beautifulsoup4

# Recupero automatico codici sconto
git clone https://github.com/shopilo-it/codice-sconto-[negozio]
python fetch.py
